\section*{Abstract}

This project ran four AI tools in sequence to create a knowledge artifact on Privacy and Data Protection in AI systems,
a core topic from Neeli Prasad's IEEE Distinguished Lecture on Human-AI Collaboration.
The workflow was defined in the following steps: Perplexity gathered research on privacy risks and regulatory frameworks,
then Claude API synthesized the findings into a 6 pages article,
after that QuillBot rewrote the content in different styles to compare clarity, 
and finally, SlidesAI.io generated a 9-slide presentation from the curated material.
Every prompt, configuration, and output is documented so the process can be reproduced and examined.

Claude API produced the clearest technical writing.
QuillBot highlighted how changing vocabulary affected readability.
SlidesAI.io created slides with no manual design work needed.
The free-tier versions of these tools did impose limits: QuillBot's 125-word paraphrase cap and SlidesAI.io's template rigidity.
