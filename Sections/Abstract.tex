\section*{Abstract}

This project ran four AI tools in sequence to create a knowledge artifact on Privacy and Data Protection in AI Systems, a core topic from Neeli Prasad's IEEE Distinguished Lecture on Human-AI Collaboration.
The workflow was: Perplexity gathered research on privacy risks and regulatory frameworks.
Claude API synthesized the findings into a 1800-word article.
QuillBot rewrote the content in different styles to compare clarity.
SlidesAI.io generated a 9-slide presentation from the curated material.
Every prompt, configuration, and output is documented so the process can be reproduced and examined.

Claude API produced the clearest technical writing.
QuillBot highlighted how changing vocabulary affected readability.
SlidesAI.io created slides with no manual design work needed.
The free-tier versions of these tools did impose limits: QuillBot's 125-word paraphrase cap and SlidesAI.io's template rigidity.
This report documents the full methodology, compares tool quality across their different roles, and reflects on what it means to use multiple AI systems responsibly in academic work.
The approach mirrors Prasad's call for transparency: showing all the steps, not just the final product.
